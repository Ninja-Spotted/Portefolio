# ESP32 as a IED for for Protection, Automation and Control  

## Functionalities builtin:
* 32 bit CPU with 2 cores (Clock: 240 MHz)
* 16 I/O Ports (at least)
* Wi-fi and USB
* RAM: 520 Kbytes
* Flash: 4 Mbytes
